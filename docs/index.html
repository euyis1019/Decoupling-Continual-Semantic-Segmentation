Abstract

Continual Semantic Segmentation (CSS) requires learning new classes without forgetting previously acquired knowledge, addressing the fundamental challenge of catastrophic forgetting in dense prediction tasks. However, existing CSS methods typically employ single-stage encoder–decoder architectures where segmentation masks and class labels are tightly coupled, leading to interference between old and new class learning and suboptimal retention–plasticity balance.

We introduce DecoupleCSS, a two-stage framework for CSS. By decoupling class-aware detection from class-agnostic segmentation, DecoupleCSS preserves past knowledge while learning new classes. Stage one fuses pre-trained text and image encoders (adapted via LoRA) to encode class-specific information and generate location-aware prompts. Stage two employs the Segment Anything Model (SAM) to output precise segmentation masks, ensuring segmentation knowledge is shared across both new and previous classes.

Method Overview

1 Class-Aware Detection

  • Text–Image Fusion: extract class-specific semantics
  • LoRA Adaptation: adapt image encoders while preserving old knowledge
  • Prompt Generation: produce class- and location-aware prompts

2 Class-Agnostic Segmentation

  • SAM Integration: precise masks via class-agnostic foundation model
  • Knowledge Sharing: masks shared across old/new classes
  • Consistent Performance: uniform quality for all classes

Architecture Details

Pre-trained Text Encoder

Processes class-specific textual cues to guide detection.

LoRA-adapted Image Encoder

Efficient adaptation with minimal forgetting.

Prompt Generator

Creates location-aware prompts from fused features.

SAM Segmentator

Generates high-quality class-agnostic masks.

Experimental Results

Pascal VOC 2012

Improvements on 19-1, 15-5, 10-1, 2-2, and 4-2; better retention and robust in challenging splits.

ADE20K

Strong performance on 100-50 and 150-50 multi-class settings.

Challenging Scenarios

Robustness under 2-2 and 4-2 incremental regimes.
